# 武浩个人简历主页

## 项目概述
深色科技风单页个人网站，支持中英文切换，部署于 GitHub Pages。

## 功能特性
- 星空粒子背景（三层视差 + 星云光晕）
- 鼠标粒子交互（引力吸引）
- Hero 打字机循环效果
- 滚动渐显动画
- 图片灯箱预览
- 视频嵌入
- 中/英文双语切换
- 霓虹发光 + 玻璃拟态 UI
- 移动端响应式

## 页面板块
1. **Hero** — 名字 + 打字机标语
2. **About** — 个人简介
3. **Skills** — 技能图标网格
4. **Gallery** — 图片 + 视频
5. **Projects** — 项目卡片
6. **Contact** — 社交媒体链接

## 使用方法
1. 本地开发：直接打开 `index.html` 或 `python3 -m http.server 3000`
2. 部署：`git push` 到 `SmartMous/wuhao-profile` 的 `main` 分支
3. GitHub Pages 自动部署，访问 <https://SmartMous.github.io/wuhao-profile/>
